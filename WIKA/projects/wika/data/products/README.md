# WIKA 产品表现模块

## 模块目标

优先接入以下产品级指标：

- 产品 ID
- 产品名称
- 分组 / 类目
- 在线状态
- 上架时间 / 更新时间
- 曝光
- 点击
- 点击率
- 访客
- 询盘
- 询盘率
- 近 7 天 / 30 天趋势

并基于这些指标识别：

- 高曝光低点击产品
- 高点击低询盘产品
- 长期无流量产品
- 核心引流产品
- 高转化产品
