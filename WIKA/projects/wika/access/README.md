# WIKA Access 记录

本目录用于沉淀 WIKA 接入、授权、回放与验证材料，不直接暴露敏感值。

建议先读：

1. [current-status.md](./current-status.md)
2. [validated-flow.md](./validated-flow.md)
3. [integration-notes.md](./integration-notes.md)
4. [check-results.md](./check-results.md)

这里可以存放：

- WIKA 接入现状
- 已验证流程
- 验证记录
- 经验与注意事项

这里不应存放：

- 明文 `client_secret`
- 明文 `access_token`
- 明文 `refresh_token`
