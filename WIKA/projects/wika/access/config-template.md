# WIKA 配置模板

以下只保留字段名和占位符，不写真实值。

- 账号名称：`WIKA`
- callback URL：`YOUR_CALLBACK_URL`
- redirect_uri：`YOUR_REDIRECT_URI`
- client_id：`YOUR_CLIENT_ID`
- token create URL：`YOUR_TOKEN_CREATE_URL`
- token refresh URL：`YOUR_TOKEN_REFRESH_URL`
- token 存储路径：`YOUR_WIKA_TOKEN_STORAGE_PATH`

真实值只保留在本地安全配置或 Railway Variables。
