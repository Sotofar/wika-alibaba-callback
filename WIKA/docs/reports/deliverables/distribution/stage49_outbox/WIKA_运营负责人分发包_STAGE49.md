# WIKA 运营负责人分发包 STAGE49

## 1. 发送对象

运营负责人。

## 2. 建议发送话术

请优先阅读 `WIKA_运营周报.pdf`、`WIKA_经营诊断报告.pdf` 和 `WIKA_店铺执行清单.pdf`。这些文件用于组织周复盘、拆解运营动作和安排人工补数。当前所有建议仍需人工确认，不代表已执行页面优化或店铺动作。

## 3. 应附 PDF 清单

- `WIKA_运营周报.pdf`
- `WIKA_经营诊断报告.pdf`
- `WIKA_店铺执行清单.pdf`
- `WIKA_广告分析报告.pdf`

## 4. 推荐阅读顺序

1. 运营周报。
2. 经营诊断报告。
3. 店铺执行清单。
4. 广告分析报告的补数入口。

## 5. 重点关注章节

- 订单与商品样本信号。
- 店铺执行动作。
- 广告样本缺口。
- 页面盘点缺口。

## 6. 需要采取的动作

- 分派页面盘点 owner。
- 分派广告样本导出 owner。
- 汇总业务反馈并决定下一轮报告改进优先级。

## 7. 需要补充的人工输入

- 真实广告导出样本。
- 页面人工盘点结果。
- 产品素材、规格、材质、关键词校对。

## 8. 不能误读的地方

- 当前页和已读范围不等于全量经营结果。
- 广告报告没有真实广告样本前不能判断 ROI。
- 页面优化建议不是已执行动作。

## 9. 反馈问题

1. 哪些建议可以直接进入下周执行？
2. 哪些建议缺少人工输入？
3. 哪份报告信息过多？
4. 是否需要 Excel / 飞书 / 邮件版？
