# XD 目录说明

## 目录用途

- `XD/docs/framework/`
  - XD 历史阶段文档、对照说明、权限材料、矩阵
- `XD/docs/framework/evidence/`
  - XD 脱敏证据与历史验证摘要
- `XD/scripts/`
  - XD 历史验证脚本
- `XD/projects/xd/`
  - XD 项目资料与 access 过程材料

## 当前状态

- 本轮不推进 XD 业务功能
- 本轮只允许整理 XD 相关文件目录归属，不做新的 XD API 验证或业务结论扩展

## 规则

- 今后 XD 的业务工作、文档、脚本、证据、规划材料，只在 `XD/` 对应目录内进行
- truly shared / common 内容继续保留在根目录、`shared/`、`src/` 或明确的 common 区域
- 所有中间进度、最终总结、验收结果、提交说明，一律使用简体中文输出
