- `product-analysis/`：XD 目录归档位置

- `ads/`：XD 目录归档位置
- `listing-optimization/`：XD 目录归档位置
- `inquiries/`：XD 目录归档位置
- `reports/`：XD 目录归档位置
- `action-items/`：XD 目录归档位置

## 调用规则

- 通用基础设施继续引用根目录 `shared/`
- XD 业务资料只落在 `XD/projects/xd/`
- 不复用 WIKA 的业务数据文件
