# XD Access 记录

本目录只承接 XD 历史接入材料与目录归档。本轮不推进 XD 业务验证。

建议先读：

1. [pending-checklist.md](./pending-checklist.md)
2. [setup-template.md](./setup-template.md)
3. [authorization-plan.md](./authorization-plan.md)
4. [validation-template.md](./validation-template.md)

这里可以存放：

- XD 待接入准备项
- XD 授权执行计划
- XD 历史验证记录

这里不应存放：

- WIKA 的明文 `token`
- WIKA 的明文 `secret`
- 跨账号复用的敏感配置
