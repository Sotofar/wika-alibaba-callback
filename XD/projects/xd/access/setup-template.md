# XD 接入配置模板

以下字段待填：

- 账号名称：`XD`
- 应用名称：`YOUR_XD_APP_NAME`
- client_id：`YOUR_XD_CLIENT_ID`
- client_secret：`YOUR_XD_CLIENT_SECRET`
- callback URL：`YOUR_XD_CALLBACK_URL`
- redirect_uri：`YOUR_XD_REDIRECT_URI`
- token create URL：`YOUR_XD_TOKEN_CREATE_URL`
- token refresh URL：`YOUR_XD_TOKEN_REFRESH_URL`
- token 存储路径：`YOUR_XD_TOKEN_STORAGE_PATH`

注意：

- 这些值只能写到 XD 项目层或安全配置，不写到共享层
- 不允许复制 WIKA 真实值
