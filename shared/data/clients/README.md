# 数据源客户端层说明

## 当前目标

本目录用于约束“从哪里读业务数据”这件事的优先级，不在当前阶段直接写死阿里国际站未知业务接口。

## 数据源优先级

### A. 官方接口

优先级最高。

适用条件：

- Alibaba 官方开放平台明确提供该业务接口
- 当前 token 能覆盖该接口权限
- 返回值结构可稳定复用

### B. 页面请求接口

第二优先级。

适用条件：

- 官方接口未找到或未覆盖
- 后台页面实际通过 XHR / fetch 请求加载数据
- 可在只读条件下复用这些页面接口

### C. 后台导出文件

第三优先级。

适用条件：

- 页面支持导出数据
- 导出结构稳定
- 适合日报/周报/分析任务

### D. 浏览器自动化只读采集

最后兜底。

适用条件：

- 官方接口和页面请求都不可用
- 但后台页面能稳定展示关键数据

不建议作为默认方案，维护成本较高。
