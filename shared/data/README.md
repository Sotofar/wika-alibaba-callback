# 共享业务数据层说明

## 模块职责

`shared/data/` 用于沉淀国际站业务数据读取层的通用框架，服务于：

- WIKA
- XD
- 后续新增账号

这里放的是：

- 数据源优先级规则
- 通用字段标准
- 通用校验规则
- 报告模板
- 建议生成规则

这里不放：

- 任一账号的真实 token
- 任一账号的真实数据结果
- 任一账号的生产配置

## 当前阶段定位

本轮先完成第一阶段骨架：

- 明确真实数据源优先级
- 为店铺概览、产品表现建立第一版模块结构
- 为管理层简版报告、运营执行版报告建立模板
- 为建议生成建立第一版规则框架

## 子目录说明

- `clients/`：数据源类型与优先级约定
- `normalizers/`：未来统一标准化逻辑的放置位置
- `field-maps/`：未来跨账号通用字段映射规则
- `validators/`：未来通用数据校验规则
- `report-templates/`：共享报告模板
- `recommendation-rules/`：共享建议生成规则
