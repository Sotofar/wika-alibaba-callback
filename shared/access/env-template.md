# 环境变量模板

以下只写模板，不写真实值。

```env
APP_BASE_URL=YOUR_APP_BASE_URL

ALIBABA_CLIENT_ID=YOUR_CLIENT_ID
ALIBABA_CLIENT_SECRET=YOUR_CLIENT_SECRET
ALIBABA_REDIRECT_URI=YOUR_CALLBACK_URL
ALIBABA_AUTH_URL=YOUR_AUTH_URL
ALIBABA_TOKEN_URL=YOUR_TOKEN_CREATE_URL
ALIBABA_REFRESH_TOKEN_URL=YOUR_TOKEN_REFRESH_URL
SESSION_SECRET=YOUR_RANDOM_SESSION_SECRET

ALIBABA_PARTNER_ID=YOUR_PARTNER_ID_IF_NEEDED

ACCOUNT_TOKEN_STORAGE_PATH=YOUR_TOKEN_STORAGE_PATH
ACCOUNT_AUTO_REFRESH_ENABLED=true
ACCOUNT_REFRESH_BUFFER_SECONDS=600
ACCOUNT_BOOTSTRAP_REFRESH_TOKEN=YOUR_REFRESH_TOKEN_IF_BOOTSTRAP_IS_NEEDED
```

## 使用规则

- 共享文档只保留变量名和占位值
- 真实值只写到本地安全配置或 Railway Variables
- 不同账号使用不同 token storage path
